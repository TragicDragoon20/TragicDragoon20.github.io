Project Details

This was a solo project submitted for my COMP270: Mathematics for 3D Worlds & Simulations module in second year. The goal of this module was to learn the fundamental mathematics involved in the design and development of real-time 3D worlds and simulations. For this module we were given 4 worksheets to complete with the first 3 being more specific in requirements whilst the 4th was more open with VFX/mesh manipulation involved.

For the 4th worksheet the effect I created was a tornado which is created from a cylinder mesh that was manipulated. Whilst also creating a shader graph to add additional effects to the look of the tornado.

Mesh Generation

For the tornado itself it is made with two meshes. One of these meshes contains the larger mesh which is used for the wind material and the smaller is used for the colouring of the tornado.

These meshes are build using a mesh builder script which adds all the triangles, vertices, normals and UVs to the mesh after they have been calculated. These are calculated through the Proceduralbase script which has a function that will build rings which are used to create each section of the tornado and connect them together to make it one solid mesh. This also allows the user to change the height, radius and number of segments within the mesh that is generated.

Shader Graph

To make the shader for the tornado I used some vertex manipulation in order to give the tornado a wobble effect to help play into the fact that the tornado is moving and not a solid object that goes from the top to the ground.

In addition to this I utilised some simple noise to create the tornado twirling effect.

What went well

For this I was happy with the way the shader turned out as it was the first time I had properly touched unity's shader graph and felt that the effect I created worked quite well for a tornado.

What could have been improved

An area I would like to improve is how I manipulated the mesh to create the larger and smaller sections as it didn't really have anything behind it except adding a random value to the ring to make it larger or smaller at different points.
