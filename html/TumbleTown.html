Project Details

The theme for the Plymouth Game Jam 2020 was 'Untouchable', with this theme we decided to make a game about a tumble weed trying to escape from towns of cacti. For this project I worked in a team of 4 people which included 2 programmers, 1 designer and 1 artist.

My Contribution

The main area I worked on for this project was the AI which uses inheritance to create multiple types of enemies. This AI was paired with some triggers to determain which enemies would be enabled to start firing on the player. The project also had a diegetic UI for the main menu which I worked on along side the designer to get the camera to move towards a different area of the map when wanting to see the level select.
